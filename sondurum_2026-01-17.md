## sondurum_2026-01-17

Bu dosya, projede su ana kadar yapilan isleri ozetler.

### Genel

- Proje statik HTML + CSS + JavaScript olarak calisir.
- Sayfalar:
  - `/` anasayfa (`index.html`)
  - `/dream.html?slug=...` detay
  - `/admin.html` admin panel

### SEO / Static dosyalar

- Favicon: `/img/favicon.png`
- Robots + sitemap:
  - `/robots.txt`
  - `/sitemap.xml`
  - `/sitemap.html`

### Turso (Database)

- Schema: `db/schema.sql`
- Seed: `db/seed.sql`
- Admin SQL endpoint: `POST /api/admin/sql` (INSERT/SELECT whitelist)
- Admin panel: SQL calistir bolumu + ozet metrikler

### API (Vercel serverless)

- Public:
  - `GET /api/dreams` liste/arama/filtre
  - `GET /api/dreams/:slug` detay (no-store)
  - `POST /api/dreams/:slug/view` view artirir
  - `POST /api/dreams/:slug/vote` like/dislike artirir
- Admin (token):
  - `GET /api/admin/summary`
  - `GET /api/admin/dreams`
  - `GET /api/admin/categories`
  - `POST /api/admin/sql`

### Env degiskenleri

- `TURSO_DATABASE_URL`
- `TURSO_AUTH_TOKEN`
- `ADMIN_TOKEN`

### Notlar / Kisitlar

- Unique ziyaretci sayisi DB'den gelmez; analytics gerekir.
- Like/dislike tekrar oy engeli tarayici bazli `localStorage` ile.
